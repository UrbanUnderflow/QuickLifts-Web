# Publish a daily Macra KPI snapshot from Scoreboard, Experiments, purchase logs, cancel reasons, retargeting, and AppsFlyer coverage

## Core narrative

The daily Macra KPI snapshot is derived from the existing Scoreboard, Experiments, purchase logs, cancel reasons, retargeting state, and AppsFlyer coverage before any partner-facing messaging is sent. That means the operating picture is assembled from the real source artifacts first, and only then translated into any external summary, partner update, or funnel message so the story stays grounded in the current daily read rather than stale interpretation, isolated metrics, or unsupported assumptions.

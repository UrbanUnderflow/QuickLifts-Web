# Publish a daily Macra KPI snapshot from Scoreboard, Experiments, purchase logs, cancel reasons, retargeting, and AppsFlyer coverage

## Core narrative

The daily Macra KPI snapshot must be assembled from the existing Scoreboard, Experiments, purchase logs, cancel reasons, retargeting state, and AppsFlyer coverage before any partner-facing interpretation is shared. That means the operating picture is built from the live source artifacts first, then translated into any external summary, partner update, or funnel narrative only after the current daily read is in hand. If the snapshot has not been assembled from those inputs, the partner-facing interpretation should wait.

## Messaging pillars

### 1. Scoreboard and Experiments establish the daily frame

The daily snapshot starts with **Scoreboard** and **Experiments** because they set the top-line operating frame for what is changing right now. Before any partner-facing summary is written, those two artifacts tell us whether the current day is holding, improving, or softening. If Scoreboard and Experiments are mixed, the message has to stay mixed too.

### 2. Purchase logs, cancel reasons, and retargeting state explain the trust picture

**Purchase logs** show where conversion is actually happening, while **cancel reasons** and **retargeting state** help explain where trust is breaking down after interest is created. Those artifacts keep the daily KPI snapshot from becoming a shallow scorecard. They show whether the movement is tied to real buying behavior, downstream hesitation, or follow-up that is landing too late or too vaguely.

### 3. AppsFlyer coverage keeps the snapshot honest about source context

**AppsFlyer coverage** keeps the daily read grounded in what we can actually see about source mix and attribution quality. A strong day should not automatically become a broad partner claim if the underlying coverage is partial or skewed. Keeping AppsFlyer coverage in the daily snapshot makes sure the interpretation stays specific to the traffic context the operating artifacts can really support.

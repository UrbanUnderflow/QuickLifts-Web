# Refresh active variant_a experiment results before making funnel partner messaging

## Core narrative

Funnel-partner messaging must be based on the current active `variant_a` experiment readout, not stale assumptions. Before any partner-facing funnel copy, campaign framing, or trial-start recommendation is approved, the active `variant_a` result set needs to be refreshed and read in its current state. If the readout is old, partial, or inferred from prior decisions, the message is not ready.

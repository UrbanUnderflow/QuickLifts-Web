# Heartbeat OS Definitions

## 1. Color Semantics
- **Red**: Indicates high urgency and immediate action required.
- **Green**: Signifies a smooth operational status.
- **Yellow**: Requires an alert but is not critical.

## 2. Narrative Lens Rotation
- **Health Lens**: Focuses on metrics related to physical activity and overall health.
- **Engagement Lens**: Emphasizes user interactions and feedback.

## 3. Beat Taxonomy
- **Check-in Beat**: Regular updates on team progress and individual tasks.
- **Action Beat**: Assigned tasks requiring immediate attention.

## 4. Pulse Cadences
- **Daily**: Routine checks for updates and progress.
- **Weekly**: Comprehensive review and planning sessions.

## 5. Idle Triggers
- **Timeout**: User inactivity leads to a trigger for alerts or notifications.
- **Status Change**: Notifications when an important status changes.
